# Bilibili Analyse
这是一个B站的分析器，目前阶段只有下载视频的功能，其他功能会在以后慢慢完善

作者：ZnWillbur

## 使用方法
- 拥有python环境
- 安装依赖
  ```
  python InstallDepend.py
  ```
- 在文件夹中输入以下命令即可
  ```
  python Bilibili.py
  ```

## 优点
- 番剧和视频实现了超高速下载，采用了多线程技术
- 采用了自动化登录
- 支持断点续传

## 目前的缺点
怎么说呢，主要是作者是个苦逼高中生，没时间，而且还有很多不懂的
- 没有ui界面
