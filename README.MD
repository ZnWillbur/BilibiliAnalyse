# Bilibili Analyse
这是一个B站的分析器，目前阶段只有下载视频的功能，其他功能会在以后慢慢完善

作者：ZnWillbur

## 使用方法
- 拥有python环境
- 安装依赖包
  ```
  pip install requests
  ```
- 在文件夹中输入以下命令即可
  ```
  python Bilibili.py
  ```

## 优点
- 番剧和视频实现了超高速下载，采用了多线程技术
